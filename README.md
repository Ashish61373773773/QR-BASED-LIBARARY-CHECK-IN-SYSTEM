# QR-Based Library Check-In System

A Node.js + Express + MongoDB project to manage library entry using QR codes.

## Features
- QR Card Generator for visitors
- QR Scanner for check-in/check-out
- Entry/Exit tracking
- Recent visitor logs
- Simple admin dashboard

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- QRCode npm package
- Frontend: HTML/CSS/JS
